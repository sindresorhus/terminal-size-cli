# terminal-size-cli

> Reliably get the terminal window size - cross-platform

## Install

```sh
npm install --global terminal-size-cli
```

## Usage

```
$ terminal-size --help

  Example
    $ terminal-size
    143
    24

  First line is the number of columns
  Second line is the number of rows
```
